# gpcr-tm6-analysis
Python script to calculate the distance between TM6 helix ends in GPCR structures using BioPython
- What it does: Скрипт
вычисляет расстояние между атомами СА (альфа-углеродами) на концах 6-й спирали.
- Why it matters: Смещение
TM6 — ключевой маркер активации GPCR.
- Tools: Python, BioPython, NumPy.
